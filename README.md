# 🎮 Ace Playz Gaming Store

A stunning, highly-interactive web storefront for a premium gaming cafe and game launcher. Designed with a sleek, dark-themed UI featuring fluid animations and a responsive mobile-first approach.

## ✨ Features

- **Dynamic GSAP Animations**: Buttery-smooth scroll reveals, 3D tilt effects on cards, and staggered grid loading using GSAP & ScrollTrigger.
- **Mobile-Optimized Micro-Interactions**: Custom swipe gestures for sliders and satisfying Android-style ripple effects on touch devices to ensure the site feels like a native app.
- **Intersection Observers**: Intelligent background pausing for image slideshows when scrolled out of view, ensuring zero layout shifts and maximum performance.
- **Premium UI/UX**: Neon green accents, glassmorphism navbars, and custom cursor rings for an immersive gaming aesthetic.

## 🚀 Tech Stack

- **HTML5 & CSS3**: Semantic structure with advanced CSS grid/flexbox layouts and custom media queries.
- **Vanilla JavaScript**: Lightweight, custom-built logic for slideshows, tabs, and event handling without heavy framework overhead.
- **GSAP (GreenSock)**: Industry-standard animation library for scroll-based and mouse-tracking animations.

## 📱 Responsiveness

Fully optimized for all devices, from ultra-wide gaming monitors to mobile phones. Features dedicated touch-friendly mobile layouts with custom swipe gestures for the best possible user experience.
